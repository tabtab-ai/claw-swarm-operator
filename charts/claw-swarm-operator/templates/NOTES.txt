Claw Swarm Operator has been installed in namespace {{ .Release.Namespace }}.

The operator runs as an internal controller and does not expose a public endpoint.
It manages a warm pool of OpenClaw instances automatically.

Check the operator status:
  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "claw-swarm-operator.name" . }}

View operator logs:
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "claw-swarm-operator.name" . }} -f

List managed instances:
  kubectl get statefulsets -n {{ .Release.Namespace }} -l tabtabai.com/tabclaw=true

Allocate an instance:
  kubectl label statefulset <name> -n {{ .Release.Namespace }} tabtabai.com/tabclaw-occupied=true

Free an instance:
  kubectl label statefulset <name> -n {{ .Release.Namespace }} tabtabai.com/tabclaw-occupied-
